Simplifying Caregiver Tasks:
A Web-Based Solution for Coordinating At-Home Medical Care via LevelUp Meds (WIP)

The purpose of the Level-Up Meds web app is to help people providing home care efficiently organize their efforts to provide the best care possible. The result myself and fellow developers are looking for is a web application that allows users to create a profile for patients in their care, assigning what medicines are needed for each patient. The app will include a calendar function that can track the dates and times when each medication needs to be given. Caregivers for the same recipient will be able to communicate with one another, allowing for enhanced coordination for the best quality of life outcome. This project has been worked on by a previous capstone group, so much of the framework has already been put together, but many core functions are incomplete or not implemented at all. Our goal for this project is to take the framework from the last group and expand on it to create a usable application, focusing primarily on the core functionality of the project. Specifically, we will focus on updating the app onboarding and chat functions and will be implementing the calendar, establishing a minimum viable product that’s publicly testable.
