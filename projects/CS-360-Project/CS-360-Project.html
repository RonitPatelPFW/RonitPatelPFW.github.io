CS 360 Project: Keep Body and Soul Togethor-er

Project Summary

Github

This game was created for my software development class. In the class, we were tasked to develop a project through the course of an entire semester. To help with such a large project, we were split up into different teams, with this project recieving a total of 5 members.


A minigame-centered tool to teach students how various data structures function. The student performs the roles of the data structure: inserting data, sorting the structure, and finding data. After playing, students have a visual understanding of the strengths and weaknesses of various data structures.

      - Learned C# and basic Unity functionality
      - Created a system to dynamically find and categorize every minigame at runtime
      - Performed playtesting and fixed bugs
      - Documented functionality and known bugs for future developers

For our project, we knew that we wanted to create a video game. However, one of the requirements was to make it computer science themed. To do this, we decided on a minigame based educational game that taught players how various data structures operated. We took heavy inspiration from the Mario Party and Survivor franchises in how we envisioned the game play. We decided to attempt to implement the following features:

      - Interactive Minigames
      - Dynamic Menus
      - Menus
      - Multiplayer
      - Gameboard

We divided the group as follows, each person took one goal to work on:

      Multiplayer: Me
      Interactive Minigames and Save Data: Neal Birchfield
      Dynamic Menus: Bricks Hudson
      Menus and Sound: Eric Minic
      Gameboard: Tyler Delegrange
