Blackjack
- A 2D Blackjack game where players are playing against the computer
  - Technology used: Java, CSS and JavaFX
  - Using JavaFX I developed a GUI that would be interactable where a player could either hit or stand against the dealer bot.
  - Cards would display for both the Dealer and Player for each hit they decide.

Paint
- A basic Paint Application where players can draw a variety of shapes and includes other features as well.
  - Developed in Java, CSS and JavaFX
  - I developed a GUI in which there would be three sections
    - Tool, Color and a drawing portion
  - In the tool menu, a user could decide on either draw, move, or delete a shape of their choosing
  - For the color menu, there is an RGB slider and a color preview
  - The project was designed to help understand the significance of OOP features like Polymorphism and Abstraction
